# Data Analysis Workflow

## Goal

Use AI tools to accelerate analysis planning while keeping numerical work verifiable.

## Workflow

1. Define the business or product question.
2. Draft an analysis plan.
3. Clean and inspect the data manually.
4. Use AI for code scaffolding or explanation.
5. Validate outputs with checks and charts.

## Output

- Reproducible analysis notes.
- Charts or dashboard preview.
- Business recommendation summary.
