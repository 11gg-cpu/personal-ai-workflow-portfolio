# Job Search Workflow

## Goal

Use AI tools to make internship applications more structured without outsourcing judgment.

## Workflow

1. Analyze the job description.
2. Extract requirements and keywords.
3. Map personal evidence.
4. Rewrite bullets with accurate claims.
5. Manually review tone, facts, and role fit.

## Output

- Tailored resume bullets.
- Interview talking points.
- Application tracker notes.
